# Style Guide
Assuming that we are using Python for this project set up Black as a code formatter:
https://black.readthedocs.io/en/stable/the_black_code_style/current_style.html

Setup Guide:
https://www.youtube.com/watch?v=n8EtFlaSRVM

To keep the code consistent we should open a new branch every time we go to solve a new issue that was on the project board and then merge the branch to main once the code has been checked.

Any functions we create should have a comment saying which requirement it fulfils to keep everything organized.
